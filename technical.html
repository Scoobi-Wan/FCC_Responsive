Strings

A string refers to a sequence of characters, meaning words like "baboon" or "lasagna" or a name like "Martha Stewart" can be a string.

A string could also be an entire sentence, or a novel if you really wanted!

Even "DEHGKJDEHGKHJedkjehgdkjehgkegdkghdkhg" is a string! Radical!

var guestName = "Martha Stewart";
var meal = "Spaghetti and Meatballs";

console.log("Hello, " + guestName + "! Welcome to Paulie's Pasta Palace. Would you like to try our famous " + meal + "?");

Output: Hello, Martha Stewart! Welcome to Paulie's Pasta Palace. Would you like to try our famous Spaghetti and Meatballs?

As seen above, strings can be concatenated together using the '+' operator.

Numbers

JavaScript keeps things simple, there is no differentiation between floats, ints, doubles, longs. Just Numbers.

var num1 = 5;
var num2 = 8.22;

Just think of all the cool things you can do with numbers!

Booleans

Just like any other language, booleans can only be true or false

var ageRequirement = 21;
var age = 19;
var oldEnough = age >= ageRequirement;

Output: true

Just think of all the cool things you can do with Booleans!

Arrays

Arrays hold sequences of data types, and can be indexed. They are declared with brackets, with the data types separated by commas.

var myCats = ["Albert", "Carl", "Jax", "Taco"];
myCats.push("Stripes");
console.log(myCats[0]);
console.log(myCats[4])

Output: Albert
Stripes

Arrays can also hold mixed data types!

Objects

Objects can be used to represent so many different things that it is impossible to cover here, but the basic gist is an Object represents something, which has:

  • attributes
  • possibly methods

    • var Carl = {name:"Carl", hairColor:"black", age:"2", weight:"9"}
    console.log(Carl.name + " is " + Carl.age + " years old." );

    Output: Carl is 2 years old.

    Objects are a very powerful data type!

    Null & Undefined

    Null and Undefined are similar, and yet different.

  • Undefined refers to a variable without a value.
  • Null refers to a variable that does not exist. It can be used to "clear" a variable.
  • It should be noted that Null and Undefined are equal in value (null == undefined evaluates to true)
    but different in type (null === undefined evaluates to false)

    • var myDogs;
    console.log(myDogs);

    Output: Undefined

    var myCats = ["Albert", "Carl", "Jax", "Taco"];
    myCats = null;

    Null and undefined are tricky!
